    View Source LAP2.Main.ConnectionSupervisor (LAP2 v0.1.0)

    Module for supervising the status of various network connections. Dispatches proxy requests via the ProxyManager. Encodes/decodes and routes data above the proxy abstraction layer. Receives commands directly from the Master module, interfaces with ProxyManager and IntroductionPoint modules. Relays are abstracted away by the ProxyManager, this module handles connections bound to proxy sequences.
