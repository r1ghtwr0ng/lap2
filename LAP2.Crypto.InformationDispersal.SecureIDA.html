    @spec disperse(binary(), integer(), integer(), integer()) :: [Share.t()]

    Encrypt and split the data into the given number of shares. Encryption keys are split using Shamir's secret sharing, while the data is split using Rabin's IDA.

    arguments

     Arguments

    • data - The data to split (binary)
    • n - The number of shares to generate (non-negative integer)
    • m - The size of each share (non-negative integer), also the reconstruction threshold

    returns

     Returns

    • A list of shares, each containing the share data and its index (list(Share.t()))
