    API Reference LAP2 v0.1.0

    modules

     Modules

    LAP2

    The main module of the LAP2 server. This module is responsible for starting the supervision tree, loading the config and setting the log level.

    LAP2.Crypto.CryptoManager

    Module for managing the state of cryptographic information for proxy connections.

    LAP2.Crypto.Helpers.CryptoStructHelper

    Contains helper functions for various cryptographic ops

    LAP2.Crypto.InformationDispersal.RabinIDA

    Rabin's Information Dispersal Algorithm, used to split and reconstruct data.

    LAP2.Crypto.InformationDispersal.SecureIDA

    Security enhanced Information Dispersal Algorithm, used to split and reconstruct data.

    LAP2.Crypto.Padding.PKCS7

    PKCS7 padding and unpadding.

    LAP2.Main.Helpers.ProcessorState

    Helper functions for processing the share handler state.

    LAP2.Main.Helpers.ProxyHelper

    Contains helper functions for processing common proxy requests and responses

    LAP2.Main.Master

    Module for sending/receiving data via the network.

    LAP2.Main.ProxyManager

    Module for handling ProxyManager requests.

    LAP2.Main.StructHandlers.RequestHandler

    Module for handling reconstructed requests from the Share Handler. Route to Proxy or Master module.

    LAP2.Main.StructHandlers.ShareHandler

    Handle share caching and reassembly.

    LAP2.Math.Matrix

    Module for performing mathematical matrix operations. The functions used for performing the vandermonde matrix operations are based on the follwoing repository: https://github.com/mmtan/IDA

    LAP2.Networking.Helpers.OutboundPipelines

    Contains outbound pipelines for dispersing and sending requests and responses.

    LAP2.Networking.Helpers.RelaySelector

    Contains functions for splitting up data into shares, then sending them via the appropriate relays.

    LAP2.Networking.ProtoBuf
    LAP2.Networking.Resolver

    This module contains helper function for resolving addresses.

    LAP2.Networking.Router

    Stores routing information, updates DHT and stores information for resolving addresses. The actual functions for resolving the addresses, extracting keys, etc. is located in LAP2.Networking.Resolver This module also stores connection information (anonymous paths and corresponidng crypto keys) TODO have a look at a secure way of storing the keys

    LAP2.Networking.Routing.Local

    Module for routing cloves to the local share handler.

    LAP2.Networking.Routing.Remote

    Module for routing cloves to remote destinations.

    LAP2.Networking.Routing.State

    Module for managing the state of the routing module.

    LAP2.Networking.Sockets.Lap2Socket

    Module for serialising/deserialising ProtoBuf cloves and sending/receiving them over UDP.

    LAP2.Networking.Sockets.UdpServer
    LAP2.Utils.ConfigParser

    Parses a JSON file into a map.

    LAP2.Utils.EtsHelper

    Common helper functions for working with ETS tables.

    LAP2.Utils.JsonUtils

    A collection of functions for working with JSON.

    LAP2.Utils.ProtoBuf.CloveHelper

    Helper functions for generating clove information, checksums, splitting and reconstructing cloves, padding, etc. Handles serialising and deserialising cloves (simplifies serial data types).

    LAP2.Utils.ProtoBuf.RequestHelper

    Helper functions for processing LAP2 requests. Verifies integrity, serves as interface for serialising/deserialising with ProtoBuf.

    LAP2.Utils.ProtoBuf.ShareHelper

    Helper functions for processing IDA Shares. Verifies integrity, serves as interface for serialising/deserialising with ProtoBuf.
